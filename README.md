# GRAP-6TWEA

## Doel
Het doel van dit project was om de verouderde **Elector Proton Robot** te moderniseren.  
Dit heb ik gedaan door het originele PCB (Printed Circuit Board) te vervangen. De nieuwe schema’s zijn te vinden in **"schematics"**.

Daarnaast moest ik ervoor zorgen dat de nieuwe sensoren correct werken.  
Ik ben gestart met experimenteren via Arduino IDE, zie **"code_arduinoIDE"**.

Op basis van deze experimenten heb ik een bibliotheek ontwikkeld om het gebruik te vereenvoudigen:

- **Versie 1** – **"liberary/robot"**: Voor Arduino, met de originele componenten aangesloten.
- **Versie 2** – **"liberary/robot_ESP_addon"**: Voor de ESP32-C6 met de bijbehorende uitbreiding.
